等差数列求和

从高斯的故事开始

问题 经典

求等差数列前 n 项和：Sₙ = a₁ + a₂ + a₃ + ... + aₙ

其中公差为 d，即 aₖ = a₁ + (k-1)d

01

高斯配对法

将数列正序与倒序相加：

Sₙ = a₁ + a₂ + ... + aₙ
Sₙ = aₙ + aₙ₋₁ + ... + a₁
2Sₙ = n(a₁ + aₙ)
Sₙ = n(a₁ + aₙ)/2

例：1+2+3+...+100

S = 100×(1+100)/2 = 5050

直观 对称
02

通项公式法

利用通项 aₖ = a₁ + (k-1)d 展开求和：

Sₙ = Σ(a₁ + (k-1)d)
= n·a₁ + d·Σ(k-1)
= n·a₁ + d·n(n-1)/2

例：3+5+7+9+11（a₁=3, d=2, n=5）

S = 5×3 + 2×5×4/2 = 40

代数 展开
03

数学归纳法

证明 Sₙ = n(a₁ + aₙ)/2：

1. 当 n=1 时，S₁ = a₁，成立
2. 假设 n=k 时成立
3. n=k+1 时，
Sₖ₊₁ = Sₖ + aₖ₊₁ = ... = (k+1)(a₁+aₖ₊₁)/2

验证：n=3, a₁=2, d=3

S₃ = 2+5+8 = 15 = 3×(2+8)/2 ✔

严谨 证明

等比数列求和

指数增长的奥秘

问题 重要

求等比数列前 n 项和：Sₙ = a₁ + a₁r + a₁r² + ... + a₁rⁿ⁻¹

其中公比为 r，即 aₖ = a₁rᵏ⁻¹

01

错位相减法

经典方法：将原式乘以公比后相减

Sₙ = a₁ + a₁r + a₁r² + ... + a₁rⁿ⁻¹
rSₙ = a₁r + a₁r² + ... + a₁rⁿ
Sₙ(1-r) = a₁(1-rⁿ)
Sₙ = a₁(1-rⁿ)/(1-r)

例：1+2+4+8+16（a₁=1, r=2, n=5）

S = 1×(1-32)/(1-2) = 31

经典 消项
02

递推公式法

利用递推关系求解：

Sₙ = Sₙ₋₁ + aₙ = Sₙ₋₁ + a₁rⁿ⁻¹
特征方程：x = rx + a₁
齐次解 + 特解 = a₁(1-rⁿ)/(1-r)

例：3+6+12+24（a₁=3, r=2, n=4）

S₄ = 3×(1-16)/(1-2) = 45

递推 差分
03

无穷级数法

当 |r| < 1 时，无穷级数收敛：

S∞ = limₙ→∞ Sₙ = a₁/(1-r)
Sₙ = S∞ - a₁rⁿ/(1-r)
= a₁(1-rⁿ)/(1-r)

例：1 + 1/2 + 1/4 + 1/8（无穷项）

S∞ = 1/(1-1/2) = 2

极限 收敛

自然数平方和

从平面到空间

问题 进阶

求前 n 个自然数的平方和：Sₙ = 1² + 2² + 3² + ... + n²

01

立方差公式法

利用恒等式 (k+1)³ - k³ = 3k² + 3k + 1

Σ[(k+1)³ - k³] = 3Σk² + 3Σk + Σ1
(n+1)³ - 1 = 3Sₙ + 3n(n+1)/2 + n
Sₙ = n(n+1)(2n+1)/6

例：1²+2²+3²+4²+5²

S = 5×6×11/6 = 55

恒等式 裂项
02

组合恒等式法

利用组合数与平方的关系：

k² = 2C(k,2) + C(k,1)
Sₙ = Σk² = 2ΣC(k,2) + ΣC(k,1)
= 2C(n+1,3) + C(n+1,2)
= n(n+1)(2n+1)/6

例：1²+2²+3²+4²

S = 4×5×9/6 = 30

组合 计数
03

几何直观法

三维立方体堆叠的体积计算：

将 n³ 分解为三层金字塔
n³ = Σ(2k-1)k（k=1..n）
求和得：Sₙ = n(n+1)(2n+1)/6

例：三维空间中的平方和

可视化证明：立方体体积分解

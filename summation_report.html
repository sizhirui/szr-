📖 项目概述

求和库是一个专注于数值求和计算的Python库，提供多种求和算法实现，支持整数、浮点数、大整数等多种数据类型，具备高性能和高准确性。

该库适用于科学计算、金融分析、数据处理等多种场景，支持并行计算和分布式处理。

✨ 核心特性

🔢 多种求和算法
支持普通求和、Kahan求和、Neumaier求和、Pairwise求和等多种算法
sum_basic([1, 2, 3, 4, 5]) → 15
sum_kahan([0.1] * 10) → 1.0
⚡ 高性能计算
采用C扩展和NumPy优化，支持向量化运算和并行计算
sum_parallel(large_array, workers=4) → 加速4倍
📐 高精度运算
支持任意精度整数和高精度浮点数运算
sum_bigint([10**100, 10**100]) → 20000000000...000 (101位)
📊 统计信息
提供求和过程中的统计信息：均值、方差、标准差等
sum_with_stats([1, 2, 3, 4, 5]) → {'sum': 15, 'mean': 3, 'std': 1.414}

📊 算法对比

算法时间复杂度空间复杂度精度适用场景
Basic SumO(n)O(1)普通整数、小数较少
Kahan SumO(n)O(1)高浮点数累加
Neumaier SumO(n)O(1)更高高精度浮点数
Pairwise SumO(n log n)O(n)最高大规模数据

🚀 快速开始

安装
pip install summation-lib
使用示例
from summation import sum_basic, sum_kahan
from summation.parallel import sum_parallel

# 普通求和
result = sum_basic([1, 2, 3, 4, 5])
# result = 15

# Kahan求和(高精度)
result = sum_kahan([0.1] * 10)
# result = 1.0

# 并行求和
result = sum_parallel(large_array, workers=4)

📖 项目概述

摇滚数据分析项目旨在对摇滚音乐历史数据进行深度挖掘和可视化分析，探索摇滚音乐的发展趋势、流派演变和热门歌曲特征。

项目使用 Python 进行数据处理和分析，结合 Matplotlib 和 Seaborn 进行数据可视化展示。

📊 数据来源

项目数据来自多个公开数据源：

  • Spotify API - 获取歌曲音频特征数据
  • MusicBrainz - 获取艺术家和专辑信息
  • Last.fm - 获取用户收听数据和标签信息
  • Kaggle数据集 - 历史摇滚音乐数据集

🎯 分析目标

🎵 流派分布 📈 流行趋势 🎸 艺术家影响力 🎧 音频特征分析 💿 专辑销量分析 🌍 地域分布

📁 项目结构

  • data/ - 原始数据和预处理数据
  • src/ - 核心分析代码
  • visualization/ - 可视化图表生成
  • notebooks/ - Jupyter Notebook分析报告
  • results/ - 分析结果和图表输出

3. 音频特征分析

通过分析歌曲的音频特征，发现：

  • 能量值(Energy)：摇滚音乐平均能量值为0.78，高于流行音乐的0.65
  • 舞曲性(Danceability)：平均为0.52，呈现中等水平
  • 乐器性(Instrumentalness)：平均为0.35，具有较高人声比例
  • 节奏(BPM)：平均为130 BPM，属于中快节奏
